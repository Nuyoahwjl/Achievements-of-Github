# 🏅 显示成就 🏅

#### 显示成就在你的个人资料上是完全可选的；默认情况下，它们可以被任何查看你公共个人资料的人看到。

#### 你可以通过访问你的 [个人资料设置](https://github.com/settings) 来选择不显示成就。
